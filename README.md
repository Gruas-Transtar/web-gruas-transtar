🌐 Gestión del Proyecto: Grúas Transtar
Este repositorio contiene la interfaz web y los módulos de lógica para la administración de servicios de asistencia vial.

1. Requisitos de Desarrollo
No necesitas instalar lenguajes pesados. Solo asegúrate de tener:

Editor de Código: Visual Studio Code (recomendado).

Navegador: Google Chrome, Brave o Firefox actualizado.

Extensión: Live Server (opcional, para ver cambios en tiempo real).

2. Estructura del Proyecto
index.html: Estructura principal de la web de grúas.

css/: Estilos y diseño responsivo.

js/: Lógica para validaciones y manejo de datos de operarios.

assets/: Fotos de las unidades de grúa y logotipos de la empresa.

3. Cómo ejecutar el proyecto
Localmente:

Clona el repositorio o descarga los archivos.

Haz doble clic en el archivo index.html para abrirlo en tu navegador.

En Producción:

El proyecto está configurado para desplegarse automáticamente al hacer push a la rama principal en GitHub, vinculada con Vercel.
